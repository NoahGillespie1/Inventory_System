Tools/Frameworks:
- HTML/CSS
- Node.js/express
- MySQL workbench

User Guide:
1. Download all pages in the repository
2. Set up/install node.js and expresslite
3. Copy schema.sql into your mySQL workbench and run it
4. enter your mySQL workbench credentials into the app.js file
5. In the bash terminal, type 'node app.js'
6. select the html file and run without debugging

CRUD Functions:
Enter initial qunanties and select 'save'. Use the '+' and '-' buttons to add or delete quantities. All data is saved to the sql server.
